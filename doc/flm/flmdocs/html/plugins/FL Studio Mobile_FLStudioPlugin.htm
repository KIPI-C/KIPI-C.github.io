FL Studio Mobile

FL Studio Mobile的插件版

FL Studio Mobile被作为FL Studio的免费插件提供，因此您可以将您的音乐制作提升到一个新的水平。

注： 从FL Studio Mobile 3开始，您必须要在插件里中加载项目，它们不再与FL Studio直接兼容。

FL Studio Mobile插件

怎么用FL Studio Mobile插件？

  1. 注意： FL Studio Mobile插件版随FL Studio 12.4一起推出。由于插件经常更新，请及时下载最新版本的插件从Looptalk。
  2. 将FL Studio Mobile的项目传给您的电脑。 请参阅本手册的分享工程部分。 第一次做这件事的时候, 请务必选择'SEND ALL' 选项。 这样子做也会把额外的采样和乐器也会传送到FL Studio插件上面。

  3. 打开FL Studio Mobile插件版，然后在FL Studio Mobile版中加载项目。
  4. FL Studio Mobile插件版将会自动同步到FL Studio中的播放列表。速度也会遵循宿主的速度
  5. 添加至少与FLM工程一样长的空Patten并且在FL Studio按播放按钮。
  6. 从现在起，您可以添加其他插件编曲了，并且还可以使用FL Studio和FL Studio Mobile来做编曲的核心环境。
  7. （可选）当然，您也可以把FL Studio Mobile的每个轨道保存为MIDI文件或把整个工程保存为MIDI文件后用FL Studio加载这些MIDI文件.
  8. 去油管看教学视频（自备梯子）

使用的技巧

您可以将FL Studio Mobile插件的工程视为这个轨道的核心部分。 分享 。 FLM的工程可以在你的电脑和移动设备之间被添加和开发。 当FLM在FL Studio作为插件时，你可以使用音质更好的乐器和更专业的效果器，因为你有一个更强大的CPU。 这样子，您就可以制作更复杂的项目。 当把复杂的项目传回给移动设备时，你可以让轨道静音来控制CPU的使用率。
