FL Studio Mobile

FL Studio Mobile 3是一个完整的音乐制作环境，它可以作为Android、iOS和Windows平台上的应用程序以及FL Studio的本地插件来工作。这样，您可以在你的移动设备上编曲，然后还可以在FL Studio导入你的flm工程并进行更进一步地编曲了。

    获取FL Studio Mobile

    FL Studio Mobile 可以运行在Android、iOS和Windows上面(APP或FL Studio的插件)。

    概述

    FL Studio Mobile被设计成可以适应各种屏幕的大小，小至手机屏，大至电脑屏。每个FL Studio Mobile 3播放列表轨道都有一条相对应的"Rack"。 这个"Rack"可以包含任意数量的乐器和效果器插件。唯一的限制就是您的CPU功率。 现在我们也有了自动化控制的轨道了，并且也支持编辑。一切就像您在用FL Studio的桌面版一样。

    Instruments

    • DirectWave [免费] - 这是一个采样播放器，并且能够十分有效地播放采样，比如钢琴，管弦乐器，吉他等的采样。您也可以去应用内部商店购买更多音源。
    • MiniSynth [免费] - 这是一个功能齐全、低CPU消耗，拥有着好听的声音的合成器这个合成器在论坛有上千个音色补丁，您也可以捏个属于自己的音色补丁。
    • GMS (Groove Machine Synth) [需内购] - 这是一个来自PC端的可以混合多种音色，拥有多个Fx通道的合成器（了解一下在PC端的GMS）。
    • SuperSaw [需内购] - 这是一个可以快速制作EDM的音色，内置了多种立体声pad和lead的强大合成器。
    • Transistor Bass [需内购] - 这是一个单声道的 bass-line风格合成器，可以用于制造有共振风格的 basslines 和lead。

    工作区

    • Home按钮 - 用于加载和保存，控制MIDI输入，设置CPU/音频，设置节拍器响度以及访问商店。
    • 音轨播放列表 - 用于排放音符，自动化管理和音频片段来制作一首歌。
    • 架子 - 用于添加乐器或者效果器。
    • 主混音台 - 用于调整电平、平衡和控制solo。
    • 编辑器 - 编辑器内容取决于音轨播放列表选取的片段，选好了片段后编辑器就会变成可以编辑片段的样子。编辑器支持钢琴窗、自动化管理和波形编辑器。
    • 键盘，打击垫，MIDI控制器 - 具体是什么就取决于轨道。如果是鼓或音符，就是钢琴窗的样子；如果是鼓点，那就是打击垫的样子

    Frequently Asked Questions

    • How do I save my project as an audio/music file? - Use the Save > MP3 or WAV on the Home Tab. Your audio file will end up in the FL Studio Mobile installation folder under ../My Tracks. If your operating system supports it, you can also choose other save locations.
    • Where can I get technical support? - Register FL Studio Mobile and visit the FL Studio Mobile support forum.
    • How do I make feature requests and suggestions? - Register FL Studio Mobile and visit the FL Studio Mobile support forum.
    • How do I get the FL Studio Plugin version of the App? - The FL Studio Mobile FL plugin is included from FL Studio 12.4 onward. Check for updates to the plugin here. You need to have a version equal to, or later than that. FL Studio for desktop includes Lifetime Free Updates, so after you buy it, you will always get the latest version for free. Just like an App!
